Referencia de la API Interna

MMM.config expone una API interna estructurada para el control dinámico de la simulación. Puede interactuar con el motor directamente desde la consola o mediante hooks de integración.

Objeto Global STATE

Este objeto mantiene el contexto de ejecución de la aplicación.

{
  engine: QueueEngine,  // Instancia del motor activo
  isPaused: Boolean,     // Estado de pausa
  speed: Number,        // Delta time multiplier (0.01 = x1)
  mode: String          // 'standard' | 'psych'
}

Métodos del QueueEngine

.step(dt)

Avanza la lógica de simulación un paso en el tiempo.

// dt: valor incremental de tiempo basado en FPS y velocidad.
STATE.engine.step(0.01);

.getStats()

Retorna el volcado completo de métricas acumuladas hasta el momento.

{
  atendidos: 45,
  abandonos: 3,
  utilization: 0.82,
  overtime: 0.05,  // Horas en tiempo extra
  avgWait: 12.4    // Minutos
}

Esquemas de Datos

Client Object
  • t: Entry time
  • type: 'PREF' | 'GEN'
  • patience: Wait limit
Server Object
  • id: Integer
  • busy: Boolean
  • timer: Service countdown
